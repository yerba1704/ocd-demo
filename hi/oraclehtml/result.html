F_SQUARE Function Signature 1

Function for square a number. The return value is the result of multiplying the parameter by itself.

Syntax

function f_square( i_value in number default 3) return number;

Parameter

ParameterDescription
I_VALUEA numeric value. The square of any non-zero real number, whether positive or negative, is always a positive number.

Examples

EXAMPLE_1

select pkg_sample.f_square from dual;

EXAMPLE_2

select pkg_sample.f_square(4) from dual;

EXAMPLE_3

begin dbms_output.put_line( 'square of 2 is '||f_square(2) ); end;
